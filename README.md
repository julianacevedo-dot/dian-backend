# Backend DIAN FINAL v5

Ajustado al flujo correcto:

1. Abrir `https://catalogo-vpfe.dian.gov.co/User/Login`
2. Seleccionar `Empresa`
3. Seleccionar `Representante legal`
4. Llenar CC y NIT
5. Clic en `Entrar`

Variables Railway:
- REPRESENTANTE_CC
- EMPRESA_NIT
- AGENTE_SECRETO_DE_DIAN
- HEADLESS=true
- DOWNLOAD_DIR=/app/downloads
